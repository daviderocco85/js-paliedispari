# JS PalieDispari

Exercises about JavaScript functions

# Target

## Palindrome  

- Ask the user to enter a word
- Create a function to determine whether the entered word is a palindrome

## Even and Odd  

- The user chooses “even” or “odd” and enters a number from 1 to 5.
- To generate a random number (also from 1 to 5) for the computer (using a function).
- To add the two numbers.
- To determine whether the sum of the two numbers is even or odd (using a function).
- To declare who won.

# Solution

The exercise solutions are provided in the corresponding main.js files located inside the 01-palindromi and 02-pari-dispari directories.

# Notes

The exercises were separated into two distinct folders to simplify execution and keep each program isolated and easier to manage.
